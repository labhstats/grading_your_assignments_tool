# grading_your_assignments_tool
Open source tool for grading your assignments using LLMs or SLMs.

## Example data
Created using OpenAI's Prism. Nearly 100% automatically.

## Likely improvements to workflow
- Improve overall cohesiveness such as: iterate through multiple folders of students, calculate final (export) summative and or formative grade. Detailed logging of model parameters, readings and evals.
- Shiny app.
- OCR / VL reading of assignment to avoid prompt injection. Secondary eval.
- Embed similarity. Tertiary eval.
